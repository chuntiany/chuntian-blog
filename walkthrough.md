# 博客系统演示与验证报告

## 概述
本文档概述了博客系统的功能和验证情况，这是一个基于 Flask (后端) 和 Vue.js (前端) 构建的全栈应用程序。

## 已实现功能

### 后端 (Flask)
- **认证模块**: 用户注册、登录、登出 (基于 Session)。
- **文章管理**: 创建、读取、更新、删除文章 (CRUD)。
- **分类管理**: 创建、读取、更新、删除分类。
- **评论管理**: 发表评论，管理员审核流程 (批准/拒绝)。
- **系统设置**: 可配置的站点设置 (如站点标题、评论开关)。
- **数据库**: 集成 MySQL 并使用 Alembic 进行数据库迁移。

### 前端 (Vue.js)
- **前台界面**:
  - **首页**: 展示文章列表。
  - **文章详情页**: 阅读文章内容并发表评论。
  - **认证页面**: 用户登录和注册。
- **后台管理**:
  - **受保护路由**: 仅限管理员访问。
  - **仪表盘**: 系统概览。
  - **管理界面**: 提供文章、分类、评论和系统设置的完整管理功能。

## 验证结果

### 自动化测试
后端测试使用 `pytest` 全部通过 (17 个测试用例)。
- `tests/test_auth.py`: 验证用户注册和登录流程。
- `tests/test_article.py`: 验证文章的增删改查操作。
- `tests/test_category.py`: 验证分类管理功能。
- `tests/test_comment.py`: 验证评论提交和审核逻辑。
- `tests/test_settings.py`: 验证系统设置的更新。

### 构建验证
前端构建使用 `vite build` 成功通过。
- 无语法错误。
- 静态资源已正确生成在 `dist/` 目录中。

## 如何运行

### 后端
1. 激活虚拟环境: `source venv/bin/activate`
2. 运行 Flask 服务器: `flask run` (默认运行在 http://127.0.0.1:5000)

### 前端
1. 进入前端目录: `cd frontend`
2. 安装依赖: `npm install`
3. 运行开发服务器: `npm run dev` (默认运行在 http://localhost:5173)
